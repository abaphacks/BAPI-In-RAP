# BAPI-In-RAP
Consume BAPI in RAP 

https://youtu.be/Wbzv7TXLmDw

Learn how to consume a Standard BAPI in SAP RAP (RESTful Application Programming Model) using a real-time implementation scenario.

Many SAP developers face situations where they need to reuse existing Standard BAPIs while developing RAP applications. 
In this tutorial, I will show you a practical example of calling a BAPI from RAP and saving data into standard SAP tables.

What you will learn in this video
1.How to consume a Standard BAPI in SAP RAP
2.Handling data from Excel upload and passing it to BAPI
3.Creating records in standard SAP tables using RAP
4.A step-by-step real scenario implementation
